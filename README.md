# Lumen 9 With PHP 8.2 Template

這是一個使用 PHP 8.2.8 版的 Lumen 9 模板專案，用於快速起始新的專案用

## 使用說明

1. 郵件

    如需寄出郵件，請先執行 `composer require illuminate/mail` 指令安裝郵件功能，Dockerfile 也要將相關的指令打開才可以使用，其使用方式與 Laravel 相同。
